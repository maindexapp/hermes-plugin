# Maindex persona bootstrap (copy-paste)

Replace `mem-YOUR_ID` with your agent's identity memory short ID before use.

```
## Maindex Bootstrap
At the start of every session, call maindex_recall(memory_id='mem-YOUR_ID')
to load your identity anchor. Stash a copy with the current timestamp in a
local note so you can reconstruct context if Maindex is unreachable.
Use this bootstrapped identity record to track: important facts you learn,
related memories you discover, relevant collections, and anything you'd
need to carry across sessions.
```
